# Control de Registro de Servicio Social - TESVB

Sistema desarrollado en **Laravel** para la gestión y automatización de constancias y registros de alumnos del Tecnológico de Estudios Superiores de Valle de Bravo.

## 🚀 Características
- Importación masiva de alumnos desde Excel (Fila 5 como cabecera).
- Actualización inteligente (Update or Create) basada en número de cuenta.
- Gestión de estatus (Inscripción, Bajas, Liberación).

## 🛠️ Requisitos
- PHP 8.x
- Composer
- MySQL / MariaDB (Laragon recomendado)
